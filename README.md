# NewsPrism

ニュースを **3つの軸（経済・社会・外交安保）** で分析し、自分の思考のかたよりに気づくためのツールです。

ローカルで動く AI（Ollama）を使うため、記事の内容が外部に送信されることはありません。

---

## できること

### ニュースフィード
複数のメディアのニュースをトピック別（外交・政治・経済など）のカラムで表示します。気になる記事をすぐに分析できます。

### 3軸分析
記事を読んで「この記事はどんな立場で書かれているか」を3つの軸でスコア化します。

| 軸 | 左（-1.0） | 右（+1.0） |
|---|---|---|
| 経済 | 市場重視・小さな政府 | 再分配・社会保障重視 |
| 社会 | 伝統・秩序重視 | 多様性・個人の自由重視 |
| 外交安保 | 抑止力・タカ派 | 対話・ハト派 |

- 感情トーン（煽り度）の検出
- あえて逆の立場からの反論を自動生成
- 複数記事をプロットで一覧比較

### メディア比較
同じニュースを複数のメディアがどう報じているかを並べて比較できます。「NHKと産経と朝日でこの事件をどう伝えているか」がひと目でわかります。

---

## 使い方

**記事を分析する**
1. ニュースフィードから気になる記事の「3軸分析」をクリック
2. または上部の入力欄に記事のURLを貼り付けて「分析」

**メディアを比較する**
1. 「メディア比較」ページを開く
2. キーワードを入力（例: 防衛費、少子化対策）
3. 同一ニュースのグループが表示されたら、比較したいグループを選択

**フィードをカスタマイズする**
- ⚙ ボタンでフィード設定を開き、表示するメディアやトピックを変更できます
- 設定はブラウザに保存されます
