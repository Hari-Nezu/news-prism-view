# claude が考える「最強の日本メディア比較メディア」像

> 関連: [[monetization-plan]] / [[news-rights-research]] / [[youtube-news-comparison-plan]]
>
> 本ドキュメントは戦略ビジョン。実装計画は別ドキュメント参照。

---

## 核となる一行

> **「報じた事実」ではなく、「報じ方の差分」と「報じない選択」を、誰でも 5 秒で読める形に変換する観測所。**

ニュース自体を売るのではなく、**ニュースの“撮られ方”そのものをデータ化したメタメディア**。日本にはまだ存在しない枠で、AllSides（米）と Ad Fontes Media（米）の中間に、**日本固有の「沈黙の文化」検出**を足した形が最強解。

---

## 構成する 4 つのレイヤー

### L1. 観測（Observation）— 取れるものは全部取る、ただし合法範囲だけ
- RSS / YouTube 公式 / 公式 API のみ。スクレイピングは絶対やらない（守りの差別化）
- テキスト + 動画字幕 + 公開時刻 + チャンネル + サムネリンク
- **保存するのは数値とベクトルだけ**。本文は揮発させる ← これが法務的な「武器」になる

### L2. 解析（Quantification）— 3 軸 + 2 メタ軸
既存の `economic / social / diplomatic` に **2 軸足す** のが「最強」化の鍵:
- **`framing`**: 主語が政府 / 市民 / 当事者のどこに置かれているか（-1 権力者視点 ↔ +1 当事者視点）
- **`latency`**: 同一イベントを **何時間遅れて報じたか**（第一報からの差分時間）

→ 「論調」だけでなく **「視点」と「速度」** を可視化できる媒体は世界にもほぼない。

### L3. 比較（Comparison）— UI の頂点
- **イベント単位の媒体マトリクス**: 横軸=媒体、縦軸=軸、セル=スコア + 動画埋め込み
- **沈黙ヒートマップ**: 何を報じなかったかが一目で分かる
- **時系列の軸ドリフト**: 「NHK の社会軸はこの 6 ヶ月で +0.4 シフトした」が観測できる
- **媒体ペア差分**: 「NHK と産経の差が最大化したイベント Top 10」みたいな対戦表

→ ここが **滞在時間とシェアの源泉**。

### L4. 説明責任（Transparency）— 信頼の源
- 全スコアに「**なぜこの値か**」のロジック説明をクリック 1 回で出す
- 採点プロンプトと評価データセットを **GitHub 公開**
- 「採点に異論ある人は PR を投げてください」で **オープンソース・ジャーナリズム** 化

→ これがあると政治クラスタからの炎上に対して **構造的に強い**。「気に入らないなら直しに来い」が言えるメディアは強い。

## 「最強」にする 3 つの非自明な選択

### ① 自分でニュースを書かない
記事を書いた瞬間、読売・朝日と同じ土俵に立たされて勝てない。**書かないことが差別化**。メタメディアは「観測者であり続けること」が信頼の源泉。

### ② アーカイブを「数値」で持つ、本文では持たない
全文検索は競合（Google News）に永遠に負ける。だが **「2022 年 X 月、社会軸が +1.0 を超えたイベント一覧」** は誰も持っていない。 **数値時系列が独自資産**になる。時間が経つほど価値が上がる唯一の構造。

### ③ ローカル計算を「ポリシー」として掲げる
クラウド LLM を使わないこと自体を **編集方針**として打ち出す:

> 「本サイトの解析は外部 API に依存しません。あなたの読んでいる記事のテキストは Anthropic / OpenAI / Google のサーバに送られていません」

これは **メディア研究者・公務員・弁護士・記者** が情報源として使う上での決定打になる。クラウド推論メディアは原理的にこの保証ができない。

## 想定オーディエンスの "刺さる順"

1. **記者・編集者**（自社の立ち位置を客観視したい）← B2B 法人プランの最深部
2. **メディア研究者・政治学者**（時系列データが欲しい）← データセット販売
3. **政治家秘書・選挙コンサル**（争点ごとの媒体地図が欲しい）← 高単価法人
4. **PR・広告代理店**（クライアント業界の論調監視）← 既存 [[monetization-plan]] C
5. **政治クラスタ・知識層** ← Pro 個人
6. **一般読者** ← 広告

→ 上から下に向かって構築すると **単価が高い顧客を取り逃さない**。[[monetization-plan]] の「広告→個人→法人」は順序が逆で、**法人モチーフを先に置いて個人プランをその下位互換に位置付ける**ほうが ARPU が上がる。

## 一言で

**「日本メディアのオシロスコープ」**。

報道を作る側ではなく、報道の波形を測る側に立つ。波形は売れる。波形の積分（時系列）はもっと売れる。波形を「測っている」事実そのものが、メディアに対する規律として機能する ── この構造を、ローカル計算と権利クリーンを盾にして個人開発の射程内に収める、というのが claude が考える最強構成。
